## Demo

https://vitorcremonez.github.io/nano-vanity/

## Instalation

Clone the repository:

```sh
git clone git@github.com:vitorcremonez/nano-vanity.git
```

Enter inside directory:
```sh
cd nano-vanity/
```

Install the packages:
```sh
npm install
```

Run the project:
```sh
npm start
```
